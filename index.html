HTML

  • HTML or Hypertext Markup Language, is used to create the structure of a webpage.
  • Element: A type of content on a webpage.
    • Text elements
    • Image elements
    • Header elements
    • List elements
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • It is possible to embed an entire stylesheet directly in the HTML file using a <style> element in the <head> element. This internal style sheet contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code. A single change to the CSS rule will apply to all tagged page elements.
  • In order for a css style sheet to take affect, an HTML <link> element must be embedded in the program. This allows the HTML file to reference the specific visual changes you're wanting to assign.
  • To create a CSS file navigate to the assets folder in the terminal and then enter "touch style.css."
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS links are heald within the <head>.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JS links are heald within the <body>.
  • It is common practice to indent the "for" loop code below the function name within the curly brackets to add visual structure to your code.
  • Functions must be called in order to execute the code block heald within them.
